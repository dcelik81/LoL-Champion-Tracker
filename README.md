# LoL Champion Rank Tracker

## Installation

Download the .exe file from [the latest release](https://github.com/dcelik81/LoL-Champion-Tracker/releases/latest). Then just click it. You can see the notification on bottom-right corner.

## Compatible Operating Systems

- Windows 10
- Windows 11 (not tested)

## Contribution

### Prerequisites

- Python 3.13 must be installed.
- A text editor of your choice.

### Setup

```sh
git clone https://github.com/username/LoL-Champion-Tracker.git
cd LoL-Champion-Tracker

python -m venv .venv

# powershell
.\.venv\Scripts\activate
# posix
source ./.venv/Scripts/activate

pip install -r requirements.txt

git checkout -b your_branch_name
# make your changes
git add .
git commit -m "your message"
git push your_branch_name
```

# Changelog

## [1.0.3] - 2026-01-16

### Added

- Translations (Turkish, English)
- Config manager added. Now users can edit settings of the application using the file: ~/.LoLChampionTracker/config.json

Example config file content:

```json
{
    "REGION": "KR", // Faker's profile region
    "SUMMONER_NAME": "Hide+on+bush-KR1", // Faker's profile
    "CHAMPION": "ryze", // Ryze as the champion to track
    "CHECK_INTERVAL": 600, // default time interval to fetch data: 10 minutes
    "LANGUAGE": "en" // default language english
}
```

### Changed

- App icon
